### How to Use This Repository

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Alonbbar6/CourseMaterial.git
   cd CourseMaterial
   ```

2. **Start with Module 1** and progress sequentially through the modules

3. **For each module:**
   - Read the reading material first
   - Complete hands-on labs or case studies (if available)
   - Take the quiz to test your understanding
   - Review examples and visualizations

4. **Choose your language:**
   - English files: Standard naming (e.g., `Module 1_ Reading.txt`)
